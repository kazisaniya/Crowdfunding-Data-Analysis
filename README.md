# Crowdfunding-Data-Analysis
Crowdfunding Analysis – SQL, Excel, Tableau &amp; Power BI dashboards

🔹 Project Overview

This project analyzes Kickstarter crowdfunding data to understand success metrics, project outcomes, and key performance indicators (KPIs). The goal is to build interactive dashboards using Excel, SQL, Tableau, and Power BI to evaluate project performance across categories, locations, and time.

🔹 Tools & Technologies

SQL (MySQL) – Data modeling, transformations, KPI calculations

Excel (Power Query, Power Pivot) – Data cleaning, star schema building

Tableau & Power BI – Interactive dashboards & visualization

GitHub – Project version control & documentation

🔹 Data Preparation

Converted epoch dates to standard datetime format

Built a Calendar table with Year, Month, Quarter hierarchy

Performed currency conversion (Goal → USD) using static rate

Created star schema data model

🔹 Key KPIs

✅ Total Projects & Outcomes (Successful, Failed, Canceled, Live)

✅ % of Successful Projects (overall, category-wise, time-wise, goal range)

✅ Top Projects by Backers and Amount Raised

✅ Average Duration (days) for successful campaigns

✅ Funds pledged vs. goal across categories & countries
